# 2CY Agent

开源、本地优先的二次元 Agent：上传一张图，让你喜欢的角色住进电脑——替你干活，陪你说话。

整个界面是一张日漫黑白线稿风的「作者原稿桌」：分镜格、对话气泡、手绘墨线、朱红印章。

**产品主页**：https://c18229039407-arch.github.io/2cy-agent/ · **下载**：[Releases](https://github.com/c18229039407-arch/2cy-agent/releases/latest)

## 快速开始（v0.8）

唯一的前置要求：安装 [Node.js](https://nodejs.org/) 18 或更高版本（下载 LTS 版一路下一步即可）。本项目零依赖，**不需要** `npm install`。

### 方式一：下载即用（推荐，不需要会 git）

1. 点本页面右上角绿色的 **Code** 按钮 → **Download ZIP**，下载后解压；
2. 双击启动脚本：
   - **Windows**：双击 `启动-Windows.bat`；
   - **macOS**：双击 `启动-Mac.command`（如果系统提示「无法验证开发者」，右键点击文件 → 选「打开」即可；若提示没有权限，先在终端里对项目目录执行 `chmod +x 启动-Mac.command`）；
   - **Linux**：终端执行 `bash start-linux.sh`；
3. 浏览器会自动打开 http://127.0.0.1:2333 。

### 方式二：命令行

```bash
git clone https://github.com/c18229039407-arch/2cy-agent.git
cd 2cy-agent
node server.mjs   # 或 npm start
```

然后打开 http://127.0.0.1:2333 。

### 首次使用

1. 点右上角**设置**，选择提供商并填入你自己的 API key（BYOK，只存你本机，不会上传给任何人）；
2. 在右栏上传角色图、填角色名，可以让模型帮你起草人设卡；
3. 开始对话——她会带着人设跟你聊，也能认真干活。

### 支持的模型提供商

通过 Anthropic 原生协议 + OpenAI 兼容协议双通道，覆盖主流模型 API：

| 提供商 | 协议 | 默认接口 |
|---|---|---|
| Anthropic（Claude） | 原生 | api.anthropic.com |
| OpenAI（GPT） | OpenAI 兼容 | api.openai.com/v1 |
| DeepSeek | OpenAI 兼容 | api.deepseek.com/v1 |

## 开发与测试

零依赖测试套件（Node 内置 test runner）：`npm test` —— 覆盖 API 冒烟、路径穿越防护、请求体限制、密钥掩码等安全回归。

## 隐私

- **本地优先**：对话记录、角色卡、长期记忆、API key 全部存在本机 `./data/` 目录，服务只监听 `127.0.0.1`；
- **BYOK**：对话请求直接从你的电脑发往 Anthropic API，本项目没有任何中间服务器；
- `data/` 已在 `.gitignore` 中，不会被提交。

## v0.8 已实现

- BYOK 配置（key 本地存储、多提供商、模型可选）
- 角色卡：上传角色图、手填或让模型起草人设（性格 / 口癖 / 说话方式 / 羁绊）
- 角色扮演对话：人设注入 system prompt，多会话本地持久化
- **三种对话模式**：
  - **随便聊聊**——轻快直接，适合日常闲聊；
  - **让我想想**（专家）——先思考再回答（Claude 走原生自适应思维链；DeepSeek-R 等推理模型自动捕获思维过程），思维过程可展开查看；
  - **交给我吧**（Agent）——调用工具干活：抓取网页、读写工作区文件（沙箱限定在 `data/workspace/`），两种协议分别走原生 tool_use 与标准 function calling，多步循环自动执行；
- **记忆与用户画像**：记忆默认开启、静默工作，管理集中在「设置 → 记忆」——画像与每条事实可见、可改、可删，可整体关闭；沉淀方式三选：自动沉淀（切换/关闭会话时自动总结，默认关闭、开关处写明消耗你的模型额度）、手动「立即沉淀当前会话」、以及 Agent 模式下她主动用 `remember` 工具记录（零额外调用）。数据只存本机 `data/memory.json`；
- **短期记忆 / 前情提要**（v0.5 新增）：对话超过约 40 条后，自动在后台把较早的消息压缩成「前情提要」，之后每轮携带提要 + 最近 24 条原文——长对话不再失忆、token 更省。默认开启，仅在超长对话中偶尔触发一次模型调用，可在「设置 → 记忆」关闭；
- **技能面板**（v0.2 新增）：左栏点选「生成代码 / 做幻灯片 / 整理文档 / 查资料」，自动切换 Agent 模式并填入任务模板，产出保存在工作区（幻灯片为浏览器可直接放映的单文件 HTML）；
- **原生联网搜索**（v0.3 新增）：设置里一键开启，使用提供商 API 侧的官方搜索工具——Anthropic（服务端 web_search，按次计费）、Kimi（内建 $web_search）、智谱 GLM（web_search 工具）；搜索由提供商执行，本项目不经手任何第三方搜索服务。其他提供商（DeepSeek、豆包、通义、OpenAI、OpenRouter、Ollama 等）在设置里填入 [Tavily](https://tavily.com) 搜索 key（有免费额度）后同样可联网搜索（v0.6 新增）
- **侧边栏 v0.4**（参考主流 AI 工作台形态重构）：顶部新建 + 会话搜索；技能与工作空间入口；会话按「今天 / 昨天 / 近 7 天 / 更早」时间分组，条目保留话数编号；底部用户区一键进入个人设置
- **MCP 扩展**（v0.7 新增）：在「设置 → 扩展」里填名称和启动命令即可接入本地 MCP 服务器（stdio 传输，零依赖实现），她在 Agent 模式下就能使用这些外部工具，工具调用同样以步骤形式展示。注意：MCP 服务是在你电脑上运行的程序，只添加你信任的命令
- **转黑白线稿**（v0.8 新增）：角色面板一键把上传的图转为黑白线稿（浏览器端 Sobel 边缘检测，零依赖、CPU 瞬时完成，与整体画风统一）
- **自定义技能**（v0.8 新增）：「设置 → 技能」添加你自己的任务模板（名称 + 模板，「____」标记空位），与内置技能并列显示在左栏
- **手机局域网接入**（v0.8 新增）：电脑上 `node server.mjs --lan` 启动，手机连同一 Wi-Fi 访问终端显示的内网地址，输入一次 6 位配对码即可使用（配对码鉴权，非本机请求一律拦截）
- 黑白线稿风界面（设计规范见 `DESIGN.md`）
