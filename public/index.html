设置

BYOK：用你自己的 API key。全部只保存在本机 data/config.json。

目前支持 Anthropic / Kimi / 智谱 GLM 的官方搜索工具，搜索由提供商侧执行、费用走你的 key（Anthropic 按次计费）。其他提供商暂无此能力，开关不生效。

告诉她你是谁——称呼和简介会进入她的记忆（system prompt），全程只存本机。

她对你的长期记忆：一段画像 + 一份事实清单，全部存在本机 data/memory.json——你随时可见、可改、可删。「沉淀」会调用一次你的模型（消耗你的 API 额度）做总结提炼。

Agent 模式干活的产出都在这里（本机 data/workspace/）。点文件名查看内容。

所有数据都在你电脑的 data/ 目录：对话（sessions/）、角色卡与形象（character.json、avatar）、长期记忆（memory.json）、配置与 key（config.json，权限 600）。服务只监听 127.0.0.1，对话请求直连模型提供商，没有任何中间服务器。
