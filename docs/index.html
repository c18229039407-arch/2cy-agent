这一话的看点

FEATURES
第一格

BYOK，你的钥匙你做主

Claude、GPT、DeepSeek、Kimi、GLM、通义、豆包、OpenRouter、Ollama…… 填入你自己的 API key 即可对话，key 只存本机。

第二格

角色卡

上传角色图、填个名字，可以让模型代笔起草性格、口癖、说话方式与羁绊设定——她从此有了人格。

第三格

三种相处方式

随便聊聊 · 让我想想 · 交给我吧

闲聊有角色感，深思有过程可看，干活时她会真的动手。

第四格

四层记忆

会话记录、长对话自动压缩的前情提要、用户画像与事实清单、跨会话沉淀——她记得你，且每一条都可见、可改、可删。

第五格

技能面板

生成代码、做浏览器里可放映的幻灯片、整理文档、查资料——点选即进入 Agent 模式，产出保存在工作区。

第六格

联网搜索 + MCP 扩展

Anthropic / Kimi / GLM 走官方搜索，其余提供商配 Tavily key 补齐；还能接入本地 MCP 服务器，让她用上更多外部工具。

三步开场

QUICK START
壱

下载解压

点上方「下载最新版」获取压缩包并解压；先安装 Node.js 18+（零依赖，无需 npm install）。

弐

双击启动

Windows 双击 启动-Windows.bat，macOS 双击 启动-Mac.command，浏览器自动打开。

参

填 key，开聊

在「设置」里选择提供商、填入你自己的 API key，上传一张她的图——这一话开始了。
